# 超市账单管理系统

超市账单管理主要用于对超市的账单进行管理，所谓账单，就是超市与供应商进行交易的凭据，超市采购部的职员、超市的部门经理是该系统的目标用户，主要模块有：

* 账单管理模块：账单录入，账单修改，账单删除
* 人员管理模块：用户的管理
* 供应商管理模块：
